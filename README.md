# Hacathon-2

Journey from Earth to Mars

This project is an immersive fantasy storytelling website that imagines the journey of Wing Commander Ravish Malhotra, an astronaut who trained for space but never got the opportunity to fly. The story blends reality with imagination, allowing users to explore a dream-like mission to Mars through scrolling, animations, and interactive elements.

The narrative begins with a reflective moment, transitions into space travel, and takes the user through launch, deep space, landing, and exploration. While it incorporates real space concepts like G-forces and orbital trajectories, the events are fictional and presented through personal dialogue to create emotional depth. The journey concludes with a wake-up moment and a tribute, emphasizing that the experience and dedication matter as much as the destination.

The design focused on storytelling first, using cinematic visuals, dark space-themed backgrounds, glowing accents, elegant typography, scroll-based animations, parallax effects, and interactive elements to immerse the user. Built with HTML, CSS, and JavaScript, the website is fully responsive and prioritizes creativity, smooth user experience, and emotional engagement.
