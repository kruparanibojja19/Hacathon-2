# Hacathon-2
This project is an immersive storytelling website that presents the imagined journey of Wing Commander Ravish Malhotra, an astronaut who trained for space but never got the opportunity to fly. Instead of a traditional informational site, the experience is designed as a narrative-driven journey where users explore the story through scrolling, animations, and interactive elements.

The concept blends reality with imagination. It begins with a reflective moment, transitions into a dream-like mission to Mars, and takes the user through key phases such as launch, deep space travel, landing, and exploration. Each stage incorporates real space concepts like G-forces, orbital trajectories, and environmental challenges, but presents them through personal dialogue to create emotional depth. The story concludes with a wake-up moment and a tribute, emphasizing that the journey and dedication matter as much as the destination.

The design process focused on storytelling first, rather than layout. The website was structured into multiple full-screen sections to guide users through a continuous narrative flow. A cinematic visual style was used, with dark space-themed backgrounds, glowing accents, and elegant typography to create a premium feel. Scroll-based animations, parallax effects, and interactive elements were carefully added to enhance immersion without overwhelming the user.

Built using HTML, CSS, and JavaScript, the project emphasizes creativity, smooth user experience, and responsive design across devices, delivering a visually engaging and emotionally impactful digital story.
